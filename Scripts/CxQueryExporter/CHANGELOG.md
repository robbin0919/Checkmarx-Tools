# 變更紀錄 - CxQueryExporter

此專案的所有顯著變更都將記錄在此檔案中。

## [1.3.0] - 2026-03-26

### 新增內容
- 整合 **Serilog** 紀錄框架，提供系統化的日誌管理能力。
- 新增 **檔案日誌輸出 (Rolling File Logging)**：
  - 日誌將自動儲存於 `logs/` 目錄中，並具備自定義輸出格式。
  - 支援 **每日滾動紀錄 (Daily Rolling)**，確保留存歷史執行紀錄。
- 強化除錯支援：
  - 加入身分認證過程與 API 請求狀態的追蹤紀錄。
  - 個別檔案匯出失敗時將記錄錯誤詳情，確保大批次任務的執行透明度。
  - 實作全域例外攔截與日誌紀錄，便於追蹤非預期錯誤的 Stack Trace。

## [1.2.1] - 2026-03-26

### 修正內容
- 修復 `System.Text.Json` (v8.0.0) 的高嚴重性安全性漏洞 (GHSA-8g4q-xg66-9fp4, GHSA-hh2w-p6rv-4g7w)，升級至 **v8.0.5**。
- 修復 `Program.cs` 中的編譯錯誤：將 `Spectre.Console` 的 `AddTask` 參數 `total` 修正為正確的 `maxValue`。

## [1.2.0] - 2026-03-26

### 新增內容
- 整合 `Spectre.Console` 函式庫，提供更豐富的 CLI 視覺化回饋。
- 新增 **動態進度條**：在匯出大量查詢腳本時，顯示百分比、處理進度、預估剩餘時間及當前處理項目。
- 新增 **配置面板**：啟動時以表格化 (Table) 顯示伺服器、帳號及輸出路徑資訊。
- 新增 **狀態旋轉提示 (Spinner)**：在進行身份認證及抓取數據時提供非同步狀態反饋。
- 優化錯誤顯示：使用樣式化的例外報告 (Exception Display) 提升問題排查效率。

## [1.1.0] - 2026-03-26

### 新增內容
- 支援命令列參數 (CLI) 模式，提升自動化執行能力（參考 `osdx` 設計模式）。
  - 新增 `-s/--server`, `-u/--user`, `-p/--pass`, `-o/--output` 參數。
  - 新增 `-h/--help` 幫助說明。
- 優化執行時的介面反饋 (UI/UX)。
  - 增加執行標頭、進度提示及顏色標示。
  - 分類顯示處理中的語言進度。
- 強化錯誤處理機制，認證失敗時會顯示詳細的伺服器回應訊息。

## [1.0.0] - 2026-03-25

### 新增內容
- 建立 .NET 8 控制台應用程式的初始專案結構。
- `CxQueryExporter.csproj`：包含 `System.Text.Json` 依賴項的專案配置。
- `Program.cs`：實作 Checkmarx CxQL 查詢腳本匯出邏輯（使用 REST API）。
  - 支援 OAuth2 身份認證（密碼授權模式 Password Grant Type）。
  - 根據程式語言與群組名稱自動建立目錄結構。
  - 對查詢名稱進行檔名清理，確保檔案系統儲存安全。
  - 支援內部伺服器忽略 SSL 憑證驗證。
- `CHANGELOG.md`：追蹤專案變更的說明文件。
